# docs/legacy

이 디렉터리는 과거 문서/스냅샷/임시 메모를 **삭제하지 않고 보관**하기 위한 아카이브입니다.

항상 최신으로 유지하는 **canonical 문서(4개)** 는 아래만 봅니다.
- `docs/PV26_PRD.md`
- `docs/PV26_DATASET_CONVERSION_SPEC.md`
- `docs/PV26_DATASET_SOURCES_AND_MAPPING.md`
- `docs/PV26_IMPLEMENTATION_STATUS.md`

정책:
- `docs/` 루트에는 canonical 문서만 남깁니다(최대 4개).
- 회의용 수치/커버리지 등 “스냅샷”은 `snapshots/`로 보관합니다.
- 실행 메모/체크리스트/리팩터링 메모는 `howto/`, `plans/`, `analysis/`로 분류합니다.
